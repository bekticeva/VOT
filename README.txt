PROJECT: Video Object tracking

DESCRIPTION:
    All code is contained in "Vot"

	Takes a video from "videos" folder as input, processes it, and saves the output in the same 		folder.
	The processed video is saved as "output.mp4".

	To process a custom video, you must provide its absolute path and pass it to ffmpeg.split() in 		Main.java
	The output folder remains the same.

HOW TO RUN:

	run Main.java

IN CASE OF ISSUES:

    In case of issues related to file locations, replace the paths in the code with corresponding 		paths in your system.
